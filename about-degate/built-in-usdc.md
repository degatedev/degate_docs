# Built-in USDC

DeGate features a Unified Multi-chain Balance system, allowing you to manage and utilize USDC across multiple networks as if they were on a single chain.

* Native In-app Bridging: Transfer USDC across chains directly within the app. There is no need to use external browsers or third-party bridges, ensuring a faster and more secure experience.
* Seamless USDC Mobility: The concept of "bridging" is made invisible. You can \
  -> Send USDC across any networks with 1 click;\
  -> Deposit USDC into Turbo Range or Simple Earn from any networks, all without the friction of cross-chain hurdles.
* Security and Efficiency: Every transfer is processed within DeGate’s secure protocol, reducing steps to save on gas fees and minimize transaction time.
