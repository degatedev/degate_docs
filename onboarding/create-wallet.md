# Create Wallet

There are a couple of ways to create your DeGate account

## **1️⃣ Create a Wallet via DeGate Mobile App (Recommended)**

[ ![download\_app\_store.png](https://hub.degate.com/common/wallet-setup-img-en/Download_AppStore.png)](https://apps.apple.com/app/apple-store/id6742168343?pt=127643354\&ct=docs\&mt=8) [ ![download\_google\_play.png](https://hub.degate.com/common/wallet-setup-img-en/Download_GooglePlay.png)](https://play.google.com/store/apps/details?id=com.app.degate\&referrer=utm_source%3Ddocs%26utm_medium%3Ddocs) [![download\_apk.png](https://hub.degate.com/common/wallet-setup-img-en/Download_Android.png)](https://download.degate.com/apps/degate-android.apk)

Choose your preferred method to create or import a wallet. The easiest way to create a wallet is using your email, follow these steps:

**Enter your email** → **Check your email inbox** → **Enter the confirmation code** → **Done**

You will be prompted to set a password to access your wallet.

![add\_wallet\_resized.png](https://hub.degate.com/common/wallet-setup-img-en/add_wallet_resized.jpeg) ![insert\_email\_cropped.png](https://hub.degate.com/common/wallet-setup-img-en/insert_email_cropped.png)&#x20;



It is recommended to use the **“Create Wallet”** and **“Import Wallet”** methods to manage high-value assets.

## 2️⃣ Create a Wallet via web app&#x20;

Visit [app.degate.com](https://app.degate.com) and click “**Connect Wallet”**, choose your preferred method to get started.
